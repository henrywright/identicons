Fun identicons for your WordPress site.

== Description ==

This WordPress plugin will generate an identicon for each user. Each identicon is highly likely to be unique.

== Installation ==

Go to Plugins > Add New and then search for "identicons".

You can then go ahead and install the plugin.

== Requirements ==

 - PHP 7.1.0 or higher
 - WordPress 3.8.0 or higher

You will need to compile PHP with the GD library for this plugin to work.

The plugin may work with earlier versions of WordPress but I haven't tested.

== Frequently Asked Questions ==

= What identicon types are available? =

Pixicon is available now. More may be added.

= How can I change the identicon type? =

Go to Settings > Discussion and then select an option under Default Avatar.

= Where should I submit bug reports? =

Please open a [GitHub](https://github.com/henrywright/identicons/issues) issue.
